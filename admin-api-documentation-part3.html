电费账单系统 - 管理员端API文档

第三部分：统计分析和系统设置

1. 统计分析

统计分析模块提供各类数据统计和分析功能。

获取控制台概览数据

GET
/api/admin/statistics/overview

获取控制台的概览数据，包括用户数、收入等统计信息

请求参数：

参数名 类型 必填 说明
timeRange string 否 时间范围（today/week/month/year）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "totalUsers": "number",     // 总用户数
        "activeUsers": "number",    // 活跃用户数
        "totalIncome": "number",    // 总收入
        "totalBills": "number",     // 总账单数
        "unpaidBills": "number",    // 未支付账单数
        "overdueRate": "number",    // 逾期率
        "userGrowth": "number",     // 用户增长率
        "incomeGrowth": "number"    // 收入增长率
    }
}

获取用电量统计

GET
/api/admin/statistics/electricity-usage

获取用电量统计数据

请求参数：

参数名 类型 必填 说明
startDate string 是 开始日期（YYYY-MM-DD）
endDate string 是 结束日期（YYYY-MM-DD）
groupBy string 否 分组方式（day/week/month）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "total": "number",           // 总用电量
        "items": [                   // 统计数据列表
            {
                "date": "string",    // 日期
                "usage": "number",    // 用电量
                "peakUsage": "number", // 峰时用电量
                "valleyUsage": "number", // 谷时用电量
                "userCount": "number"  // 用户数
            }
        ]
    }
}

2. 系统设置

系统设置模块提供系统配置、电价设置等功能。

更新电价设置

PUT
/api/admin/system/electricity-price

更新系统的电价设置

请求参数：

参数名 类型 必填 说明
peakPrice number 是 峰时电价（元/度）
valleyPrice number 是 谷时电价（元/度）
peakHours array 是 峰时时段（例：["08:00-12:00", "14:00-18:00"]）
effectiveDate string 是 生效日期（YYYY-MM-DD）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "id": "string",          // 设置记录ID
        "peakPrice": "number",   // 峰时电价
        "valleyPrice": "number", // 谷时电价
        "peakHours": ["string"], // 峰时时段
        "effectiveDate": "string", // 生效日期
        "createTime": "string",    // 创建时间
        "operatorId": "string",    // 操作人ID
        "operatorName": "string"   // 操作人姓名
    }
}

获取系统日志

GET
/api/admin/system/logs

获取系统操作日志

请求参数：

参数名 类型 必填 说明
page number 否 页码，默认1
pageSize number 否 每页数量，默认10
type string 否 日志类型（operation/system/error/all）
startDate string 否 开始日期（YYYY-MM-DD）
endDate string 否 结束日期（YYYY-MM-DD）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "total": "number",           // 总记录数
        "items": [                   // 日志列表
            {
                "id": "string",      // 日志ID
                "type": "string",    // 日志类型
                "action": "string",  // 操作行为
                "content": "string", // 日志内容
                "operatorId": "string", // 操作人ID
                "operatorName": "string", // 操作人姓名
                "ip": "string",      // IP地址
                "createTime": "string" // 创建时间
            }
        ]
    }
}
