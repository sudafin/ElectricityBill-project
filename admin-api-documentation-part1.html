电费账单系统 - 管理员端API文档

第一部分：用户管理和电表管理

1. 用户管理

用户管理模块提供用户信息的增删改查等功能。

获取用户列表

GET
/api/admin/users

获取用户列表，支持分页和筛选

请求参数：

参数名 类型 必填 说明
page number 否 页码，默认1
pageSize number 否 每页数量，默认10
keyword string 否 搜索关键词，支持用户名、姓名、手机号搜索
status string 否 用户状态筛选（active/locked/all）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "total": "number",           // 总记录数
        "items": [                   // 用户列表
            {
                "id": "string",      // 用户ID
                "username": "string", // 用户名
                "name": "string",    // 姓名
                "phone": "string",   // 手机号
                "email": "string",   // 邮箱
                "address": "string", // 地址
                "status": "string",  // 状态
                "createTime": "string", // 创建时间
                "lastLoginTime": "string" // 最后登录时间
            }
        ]
    }
}

获取用户详情

GET
/api/admin/users/{userId}

获取指定用户的详细信息

请求参数：

参数名 类型 必填 说明
userId string 是 用户ID

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "id": "string",          // 用户ID
        "username": "string",    // 用户名
        "name": "string",        // 姓名
        "phone": "string",       // 手机号
        "email": "string",       // 邮箱
        "address": "string",     // 地址
        "status": "string",      // 状态
        "createTime": "string",  // 创建时间
        "lastLoginTime": "string", // 最后登录时间
        "meters": [              // 绑定的电表列表
            {
                "id": "string",  // 电表ID
                "meterNo": "string", // 电表编号
                "location": "string" // 安装位置
            }
        ]
    }
}

添加用户

POST
/api/admin/users

添加新用户

请求参数：

参数名 类型 必填 说明
username string 是 用户名
password string 是 密码
name string 是 姓名
phone string 是 手机号
email string 否 邮箱
address string 否 地址

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "id": "string",      // 新创建的用户ID
        "username": "string" // 用户名
    }
}

2. 电表管理

电表管理模块提供电表信息的增删改查、绑定用户等功能。

获取电表列表

GET
/api/admin/meters

获取电表列表，支持分页和筛选

请求参数：

参数名 类型 必填 说明
page number 否 页码，默认1
pageSize number 否 每页数量，默认10
keyword string 否 搜索关键词，支持电表编号、用户名搜索
status string 否 电表状态筛选（active/inactive/abnormal/all）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "total": "number",           // 总记录数
        "items": [                   // 电表列表
            {
                "id": "string",      // 电表ID
                "meterNo": "string", // 电表编号
                "meterType": "string", // 电表类型
                "location": "string", // 安装位置
                "status": "string",  // 状态
                "userId": "string",  // 用户ID
                "userName": "string", // 用户名
                "lastReading": "number", // 最新读数
                "lastReadTime": "string" // 最新读数时间
            }
        ]
    }
}

手动录入电表读数

POST
/api/admin/meters/{meterId}/readings

手动录入电表读数

请求参数：

参数名 类型 必填 说明
meterId string 是 电表ID
reading number 是 读数值
readingDate string 是 读数日期（YYYY-MM-DD）
remark string 否 备注

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "id": "string",          // 读数记录ID
        "meterId": "string",     // 电表ID
        "reading": "number",     // 读数值
        "readingDate": "string", // 读数日期
        "remark": "string",      // 备注
        "createTime": "string"   // 创建时间
    }
}
