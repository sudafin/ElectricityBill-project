电费账单系统 - 管理员端API文档

第二部分：账单管理和支付管理

1. 账单管理

账单管理模块提供账单的生成、查询、修改等功能。

获取账单列表

GET
/api/admin/bills

获取账单列表，支持分页和多条件筛选

请求参数：

参数名 类型 必填 说明
page number 否 页码，默认1
pageSize number 否 每页数量，默认10
keyword string 否 搜索关键词，支持账单号、用户名搜索
status string 否 账单状态（unpaid/paid/overdue/all）
startDate string 否 账单开始日期（YYYY-MM-DD）
endDate string 否 账单结束日期（YYYY-MM-DD）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "total": "number",           // 总记录数
        "items": [                   // 账单列表
            {
                "id": "string",      // 账单ID
                "billNo": "string",  // 账单编号
                "userId": "string",  // 用户ID
                "userName": "string", // 用户名
                "meterNo": "string", // 电表编号
                "startReading": "number", // 起始读数
                "endReading": "number",   // 结束读数
                "usage": "number",    // 用电量
                "amount": "number",   // 应付金额
                "status": "string",   // 账单状态
                "billDate": "string", // 账单日期
                "dueDate": "string",  // 截止日期
                "createTime": "string" // 创建时间
            }
        ]
    }
}

生成账单

POST
/api/admin/bills/generate

根据电表读数生成账单

请求参数：

参数名 类型 必填 说明
meterId string 是 电表ID
startDate string 是 账单开始日期（YYYY-MM-DD）
endDate string 是 账单结束日期（YYYY-MM-DD）
dueDate string 是 付款截止日期（YYYY-MM-DD）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "id": "string",          // 账单ID
        "billNo": "string",      // 账单编号
        "userId": "string",      // 用户ID
        "meterNo": "string",     // 电表编号
        "startReading": "number", // 起始读数
        "endReading": "number",   // 结束读数
        "usage": "number",        // 用电量
        "amount": "number",       // 应付金额
        "status": "string",       // 账单状态
        "billDate": "string",    // 账单日期
        "dueDate": "string",     // 截止日期
        "createTime": "string"    // 创建时间
    }
}

2. 支付管理

支付管理模块提供支付记录查询、退款处理等功能。

获取支付记录列表

GET
/api/admin/payments

获取支付记录列表，支持分页和多条件筛选

请求参数：

参数名 类型 必填 说明
page number 否 页码，默认1
pageSize number 否 每页数量，默认10
keyword string 否 搜索关键词，支持支付单号、账单号、用户名搜索
status string 否 支付状态（success/failed/refunded/all）
startDate string 否 支付开始日期（YYYY-MM-DD）
endDate string 否 支付结束日期（YYYY-MM-DD）

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "total": "number",           // 总记录数
        "items": [                   // 支付记录列表
            {
                "id": "string",      // 支付记录ID
                "paymentNo": "string", // 支付单号
                "billId": "string",  // 账单ID
                "billNo": "string",  // 账单编号
                "userId": "string",  // 用户ID
                "userName": "string", // 用户名
                "amount": "number",   // 支付金额
                "paymentMethod": "string", // 支付方式
                "status": "string",   // 支付状态
                "payTime": "string",  // 支付时间
                "createTime": "string" // 创建时间
            }
        ]
    }
}

处理退款申请

POST
/api/admin/payments/{paymentId}/refund

处理用户的退款申请

请求参数：

参数名 类型 必填 说明
paymentId string 是 支付记录ID
action string 是 处理动作（approve/reject）
reason string 否 处理原因

返回数据：

{
    "code": 200,
    "message": "success",
    "data": {
        "id": "string",          // 退款记录ID
        "paymentId": "string",   // 支付记录ID
        "amount": "number",      // 退款金额
        "status": "string",      // 退款状态
        "reason": "string",      // 处理原因
        "operatorId": "string",  // 操作人ID
        "operatorName": "string", // 操作人姓名
        "createTime": "string"    // 创建时间
    }
}
